# uos-gallery

```

├── script.js           # 타이핑 애니메이션 스크립트
├── netlify.toml        # Netlify 배포 설정
├── resource/           # 이미지 리소스 (logo.png, cursor.png 등)
├── 2307/               # 2023.07.10 아카이빙 세션
│   ├── index.html      # 갤러리 목록
│   ├── 1.png           # 류채은 썸네일
│   ├── 2.jpg           # 박나현 썸네일
│   ├── 3.jpg           # 박주혜 썸네일
│   ├── 4.jpg           # 류채은 썸네일
│   ├── 1/index.html    # 류채은 - RYUFOLIO
│   ├── 2/index.html    # 박나현 - 긍정의 말들
│   ├── 3/index.html    # 박주혜 - BBSS
│   └── 4/index.html    # 류채은 - UOSVD
└── 2309/               # 2023.10.17 FAVORITE 세션
    ├── index.html      # 갤러리 목록
```

## 기술 스택

- HTML5 / CSS3 / JavaScript
- jQuery 3.6.0 + jQuery UI 1.13.2
- 폰트: Space Grotesk (Google Fonts), Pretendard
- 배포: Netlify
